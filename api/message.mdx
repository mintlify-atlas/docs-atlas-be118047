## Overview

The `Message` class represents an email message in Mailee.me. Use it to create, send, and manage email campaigns with HTML content, templates, and advanced targeting options.

## Methods

### Static Methods

#### findAll()

Retrieves all messages from your account.

```csharp
public static List<Mailee.Message> findAll()
```

**Returns:** `List<Mailee.Message>` - Collection of all messages

**Example:**
```csharp
List<Mailee.Message> messages = Mailee.Message.findAll();
foreach (var msg in messages)
{
    Console.WriteLine($"Message: {msg.Title} - Status: {msg.Status}");
}
```

---

#### findById(int id)

Retrieves a specific message by its ID.

```csharp
public static Mailee.Message findById(int id)
```

<ParamField path="id" type="int" required>
  The message ID
</ParamField>

**Returns:** `Mailee.Message` - The message object, or empty message if not found

**Example:**
```csharp
Mailee.Message message = Mailee.Message.findById(1);
Console.WriteLine($"Subject: {message.Subject}");
```

### Instance Methods

#### save()

Creates a new message or updates an existing one. If `Id` is 0, creates a new message. Otherwise, updates the existing message.

```csharp
public string save()
```

**Returns:** `string` - XML response from the API

## Usage Examples

### Create Message with HTML

```csharp
// Create Message
Mailee.Message message = new Mailee.Message();
message.Title = "Título da Mensagem";
message.Subject = "Assunto da Mensagem";
message.FromName = "Nome do Destinatário";
message.FromEmail = "Email do Destinatário";
message.ReplyEmail = "Email para Resposta";
message.Html = "<html><body>Hello World!</body></html>";
message.Analytics = true;
message.Emails = "email_teste_1@site.com, email_teste_2@site.com, email_teste_3@site.com";

// Save
message.save();
```

## Complete Example

```csharp
using System;
using System.Collections.Generic;
using Mailee;

try
{
    // Create a new message with template
    Mailee.Message message = new Mailee.Message();
    message.Title = "Weekly Newsletter";
    message.Subject = "This Week's Updates";
    message.FromName = "My Company";
    message.FromEmail = "newsletter@mycompany.com";
    message.ReplyEmail = "reply@mycompany.com";
    message.TemplateId = 5;
    message.Analytics = true;
    message.ListId = 1;
    
    // Add dynamic content to template regions
    message.Edit.Add(new KeyValuePair<int, string>(1, "<h1>Welcome!</h1>"));
    message.Edit.Add(new KeyValuePair<int, string>(2, "<p>Here are this week's highlights...</p>"));
    
    // Save and send
    string result = message.save();
    Console.WriteLine("Message created successfully");
    
    // Retrieve and check status
    Mailee.Message sentMessage = Mailee.Message.findById(message.Id);
    Console.WriteLine($"Message Status: {sentMessage.Status}");
    Console.WriteLine($"Total Recipients: {sentMessage.Total}");
}
catch (Mailee.RestException ex)
{
    Console.WriteLine("API Error:");
    foreach (string error in ex.ErrorMessages)
    {
        Console.WriteLine($"- {error}");
    }
    Console.WriteLine($"Status: {ex.StatusCode} - {ex.StatusDescription}");
}
catch (Exception ex)
{
    Console.WriteLine($"Error: {ex.Message}");
}
```

## Error Handling

All methods may throw `Mailee.RestException` on API errors:

```csharp
try
{
    Mailee.Message message = new Mailee.Message();
    message.Title = "Test Message";
    message.Subject = "Test";
    message.FromEmail = "test@example.com";
    message.save();
}
catch (Mailee.RestException ex)
{
    foreach (string error in ex.ErrorMessages)
    {
        Console.WriteLine(error);
    }
    Console.WriteLine($"Status Code: {ex.StatusCode}");
}
```
