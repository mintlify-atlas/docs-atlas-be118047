## Overview

The `List` class represents a mailing list in your Mailee.me account. Use it to create, retrieve, and update mailing lists that organize your contacts.

## Methods

### Static Methods

#### findAll()

Retrieves all lists from your account.

```csharp
public static List<Mailee.List> findAll()
```

**Returns:** `List<Mailee.List>` - Collection of all mailing lists

**Example:**
```csharp
List<Mailee.List> lists = Mailee.List.findAll();
foreach (var list in lists)
{
    Console.WriteLine($"List: {list.Name} - Contacts: {list.ListsContactsCount}");
}
```

---

#### findById(int id)

Retrieves a specific list by its ID.

```csharp
public static Mailee.List findById(int id)
```

<ParamField path="id" type="int" required>
  The list ID
</ParamField>

**Returns:** `Mailee.List` - The list object, or empty list if not found

**Example:**
```csharp
Mailee.List list = Mailee.List.findById(1);
Console.WriteLine($"List Name: {list.Name}");
```

### Instance Methods

#### save()

Creates a new list or updates an existing one. If `Id` is 0, creates a new list. Otherwise, updates the existing list.

```csharp
public string save()
```

**Returns:** `string` - XML response from the API

**Example - Create:**
```csharp
// Create List
Mailee.List list = new Mailee.List();
list.Name = "Nome da Lista";
list.Description = "Descrição da Lista";
list.Company = "Empresa";
list.Address = "Rua Teste S/N";
list.Phone = "(71) 9999-8888";
list.Site = "www.site.com";
list.Facebook = "facebook.com/seulogin";
list.Twitter = "twitter.com/seulogin";
list.Active = true;

// Save
list.save();
```

**Example - Update:**
```csharp
// Find List
Mailee.List list = Mailee.List.findById(1);

// Update Fields
list.Name = "Updated List Name";
list.Description = "Updated Description";
list.Active = true;

// Save
list.save();
```

## Complete Example

Here's a complete example showing how to create and manage a mailing list:

```csharp
using System;
using System.Collections.Generic;
using Mailee;

try
{
    // Create a new list
    Mailee.List newList = new Mailee.List();
    newList.Name = "Newsletter Subscribers";
    newList.Description = "Monthly newsletter recipients";
    newList.Company = "My Company";
    newList.Address = "123 Main Street";
    newList.Phone = "(555) 123-4567";
    newList.Site = "www.mycompany.com";
    newList.Facebook = "facebook.com/mycompany";
    newList.Twitter = "twitter.com/mycompany";
    newList.Active = true;
    
    // Save the list
    newList.save();
    Console.WriteLine($"List created with ID: {newList.Id}");
    
    // Retrieve all lists
    List<Mailee.List> allLists = Mailee.List.findAll();
    Console.WriteLine($"Total lists: {allLists.Count}");
    
    // Find and update a specific list
    Mailee.List existingList = Mailee.List.findById(newList.Id);
    existingList.Description = "Updated description";
    existingList.save();
    
    Console.WriteLine("List updated successfully");
}
catch (Mailee.RestException ex)
{
    Console.WriteLine("API Error:");
    foreach (string error in ex.ErrorMessages)
    {
        Console.WriteLine($"- {error}");
    }
    Console.WriteLine($"Status: {ex.StatusCode} - {ex.StatusDescription}");
}
catch (Exception ex)
{
    Console.WriteLine($"Error: {ex.Message}");
}
```

## Error Handling

All methods may throw `Mailee.RestException` on API errors:

```csharp
try
{
    Mailee.List list = new Mailee.List();
    list.Name = "Test List";
    list.save();
}
catch (Mailee.RestException ex)
{
    foreach (string error in ex.ErrorMessages)
    {
        Console.WriteLine(error);
    }
    Console.WriteLine($"Status Code: {ex.StatusCode}");
    Console.WriteLine($"Status Description: {ex.StatusDescription}");
}
```
