## Overview

The `Contact` class represents a contact in your Mailee.me account. It provides methods to create, retrieve, update, and manage contact subscriptions to mailing lists.

### Instance Methods

#### save()

Creates a new contact or updates an existing one. If `Id` is 0, creates a new contact. Otherwise, updates the existing contact.

```csharp
public string save()
```

**Returns:** `string` - XML response from the API

**Example - Create:**
```csharp
// Create Contact
Mailee.Contact contact = new Mailee.Contact();
contact.Email = "email@site.com";
contact.Name = "Nome";
contact.InternalId = 33;
contact.Company = "Empresa";
contact.Position = "Cargo";
contact.Sex = "M";
contact.Birthday = new DateTime(1986, 1, 30);
contact.Age = 25;
contact.Phone = "(71) 9999-8888";
contact.Mobile = "(71) 9999-8888";
contact.Address = "Rua Teste, S/N";
contact.Notes = "Anotações do Contato";

// Add Optional Fields
contact.DynamicValues.Add(new KeyValuePair<string, string>("Campo 1", "Valor 1"));
contact.DynamicValues.Add(new KeyValuePair<string, string>("Campo 2", "Valor 2"));

// Save
contact.save();
```

**Example - Update:**
```csharp
// Find Contact
Mailee.Contact contact = Mailee.Contact.findById(1);

// Update Fields
contact.Email = "newemail@site.com";
contact.Name = "Updated Name";

// Save
contact.save();
```

## Error Handling

All methods may throw `Mailee.RestException` on API errors:

```csharp
try
{
    Mailee.Contact contact = new Mailee.Contact();
    contact.Email = "email@site.com";
    contact.Name = "Test Contact";
    contact.save();
}
catch (Mailee.RestException ex)
{
    // Show error messages
    foreach (string error in ex.ErrorMessages)
    {
        Console.WriteLine(error);
    }
    Console.WriteLine($"Status Code: {ex.StatusCode}");
}
```
