## Welcome to Mailee.me .NET SDK

The Mailee.me .NET SDK provides a simple and intuitive way to integrate your .NET applications with the Mailee.me email marketing platform. Built for .NET Framework 2.0 and above, this library handles all API communication, letting you focus on building great email campaigns.

## What You Can Build

With the Mailee.me .NET SDK, you can:

- **Sync customer data** from your CRM or database to your email marketing lists
- **Automate email campaigns** triggered by user actions in your application
- **Build custom dashboards** to monitor campaign performance
- **Manage subscriber preferences** with subscribe/unsubscribe operations
- **Create dynamic email content** using custom fields and templates
